<h2 align="center">How to use 🤔</h2>

   ```
   - Repository clone:
   $ git clone https://github.com/luizeduardo29/PayFlow
   - Enter the directory:
   $ cd PayFlow
   - Install the dependencies:
   $ flutter pub get
   - Start the app: 
   $ flutter run
   ```
